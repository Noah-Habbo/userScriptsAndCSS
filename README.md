## Habbo-related userscripts and stylesheets
All userJS scripts are hosted (with auto-update) on my [GreasyFork](https://greasyfork.org/en/users/1392218-noahc500).
### Directory
| File | Use |
| ------------- | ------------- |
| [makeUsernamesMoreClickable.user.js](./makeUsernamesMoreClickable.user.js) | Adds clickable usernames on [theHabboWH.org](https://thehabbowh.org/), all linking to the EE:DB for easy checking of records. |
| [betterBillTitles.user.js](./betterBillTitles.user.js) | Changes document titles to include the name of the currently looked at bill or law (on [theHabboWH.org](https://thehabbowh.org/)), as well as for more accurate bill/law index pages. |
